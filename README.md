# spinning-inquisitor-lightsaber
Motorized spinning Inquisitor lightsaber built with Arduino, custom 3d printed parts, and addressable LEDs. 


![light up spinning shot](https://github.com/JosephDalman/spinning-inquisitor-lightsaber/blob/87917d64b4319f9299aa28a70e8a22b43614257a/LightupSpin.png)

Overview
This project is a motorized spinning Inquisitor-style lightsaber built using Arduino control, custom 3D printed mechanical components, and addressable LED technology.
The goal was to design a system that integrates mechanical motion, embedded programming, and electrical power management into a single functional prototype.
The saber blade rotates continuously while maintaining stable LED illumination, requiring careful consideration of motor torque, balance, and structural housing design.

Features
Motorized rotating blade assembly
Individually addressable LED strip control
Custom 3D printed chassis components
Embedded microcontroller-based animation logic
Prototype mechanical balancing design

Hardware Components
Microcontroller: Arduino Nano (ATmega328P)
Lighting: WS2812B addressable LED strip
Motor: High Torque 12 -24V DC motor 3500 - 7000rpm
Power Source: 3 9V alkaline batteries in series
Custom designed 3D printed structural parts


Internal drive mechanism with spur transmission system
